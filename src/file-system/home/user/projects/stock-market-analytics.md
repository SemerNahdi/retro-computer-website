**Timeline:** 11/2025

Built an end-to-end ETL pipeline for real-time stock and ETF data processing. Created interactive Power BI dashboards for financial analytics and visualization. Designed a Next.js web app with real-time streaming and background job processing using Inngest.

- **Tech Stack:** Next.js, Power BI, Inngest, ETL.
- **Key Features:** Real-time streaming, background jobs, interactive dashboards.
